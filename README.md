# QuizBuilder Application

Full-stack application designed to create, manage, and take interactive quizzes. Built with **NestJS**, **Next.js**, **Prisma**, and **PostgreSQL**.

## 🚀 Getting Started

### Local Development (Recommended)
The application is fully functional and stable when run in a local development environment.

1. **Setup Backend:**
   ```bash
   cd backend
   npm install
   npm run dev

2. **Setup Frontend:**
   ```bash
   cd frontend
   npm install
   npm run dev


    Frontend: http://localhost:3000

    Backend API: http://localhost:3001

   🐳 Docker Status
Note: The Docker configuration is currently a work-in-progress and may not be fully functional in all environments. The project is confirmed to be stable and ready for review when run locally using the instructions above.

🛠 Tech Stack
Backend: NestJS, TypeScript, Prisma ORM

Frontend: Next.js

Database: PostgreSQL

💡 Notes
Database connection settings are located in the .env file of the backend directory.

Ensure PostgreSQL is running locally if you are not using the provided Docker configuration.
